# terraform-aws

# version

```
$ terraform version
Terraform v0.11.7
+ provider.aws v1.19.0
```

# usage

1. create vars file as terraform.tfvars

  ```
  $ cat terraform.tfvars
  aws_access_key = "your access key"
  aws_secret_key = "your secret key"
  ```

2. terraform init

3. terraform apply

4. terraform destroy
